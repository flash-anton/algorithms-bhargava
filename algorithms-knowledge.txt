Скорости алгоритмов в порядке возрастания:
O(logN) - бинарный поиск.
O(N) - простой поиск.
O(N*logN) - быстрая сортировка.
O(N^2) - сортировка выбором.
O(N!) - задача о коммивояжере.

Скорость операций со структурами данных:
        Массив Список
Чтение   O(1)   O(N)
Вставка  O(N)   O(1)
Удаление O(N)   O(1)


---- 1. Бинарный поиск.
Бертся отсортированный массив.
Результат - индекс элемента в массиве, либо null.
Скорость алгоритма = O(logN).


---- 2. Сортировка выбором.
Берется неотсортированный массив.
Скорость алгоритма = O(N^2).
Делается N итераций: N сравнений + (N-1) сравнений + (N-2) + ... + 1
Или N/2 итераций по (N+1) сравнений: ((N) + (1)) + ((N-1) + (2)) + ...
Значит, сокрость алгоритма = O(N/2 * (N+1)) = O(1/2N^2 + 1/2N) = O(N^2 + N) = O(N^2)
